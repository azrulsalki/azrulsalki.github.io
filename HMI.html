HUMAN MACHINE INTERFACE

A project to replace the current inspection line of manual recording (on paper) to digital. This project is related to the data visualization project as the data are connected to calculate all the performance ouput.

-​​HTML
-CSS
-NodeJS
-PHPmyAdmin
-Wifi Router for connection

Can be access by any device that support HTML

DEFECT NOTE

All defect products are mark with their defect variable and all of them are store into the cloud database connected via wifi to the router.

MySQL Database

All database are install inside the Google Cloud PLatform (GCP).
The best thing about this database is people could access it by using XAMPP or any database that could link to the GCP IP address.
